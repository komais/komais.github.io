生物学DNA解读

标题 improved maize reference genome with single-molecule technologies

杂志 Nature

IF : 38.14

作者： Yinping Jiao

单位： Cold Spring Harbor Laboratory, Cold Spring Harbor, New York 11724, USA

性质： letter

摘要：

  • 之前组装的很多植物基因组存在着片段过多，缺少复杂重复区域的问题。
  • 使用三代，相对于之前版本，contig长度延长了52倍，对于基因间区和着丝粒区提升很大。
  • 对重复区域研究，发现了130K个完整的转座原件，可以对转座原件的谱系研究有很大帮助
  • 使用全长转录本测序，对基因组进行注释。
  • 比较基因组研究，发现两个自交系中，存在着广泛的低基因密度区域和玉米谱系特异基因的缺失

前言

  • 玉米的基因组发布于2009，有100K个contig，太碎了。

结果

组装结果

  • 组装过程如下： 组装->评估->优化contig->连scaffold->根据图谱，构建染色体->校正
  • 数据量
    • SMRT : 65X
  • 结果：
    • contig个数2958个，一半的contig长度大于1.2M，N50为1.18M，以前的只有5k
    • scaffold个数为625个，N50为9.56M
    • 使用BAC文库和遗传图谱，来构建染色体
    • 基因组大小为2.1G，只有2522个gap，估计平均gap长度为27k。

组装评估

  • 相对于以前的版本：
    • 序列相似度 99.9%
    • contig长度延长了52倍
    • 84%的BAC被一条long read覆盖。
    • 纠正了以前很多方向不对的环着丝粒区域
    • 20个arm中，有14个找到了端粒重复区域。
  • CENH3（着丝粒组蛋白H3）ChIP结果
    • 发现着丝粒很正确的安置了

      基因注释

  • 使用全长转录本优化注释。
    • 每个基因的转录本有之前的1.6个，增加到了3.3个
    • 提升了对调控区域的覆盖，降低了基因周边3k的gap，由20%降低到了1%，因此可以对TATA box，CCAAT box，Y patch motif等进行研究。调控区域的完善有助于表型的研究。

      进化分析

  • 同Sorghum（高梁）的进化分析
    • 同Sorghum（梁）分歧后，玉米经历了加倍（diploidization）和gene loss。
    • 之前研究表明gene loss只在一套基因中发生，但是我们的组装结果表明，56％的高粱直系同源基因能比对到玉米亚基因组（指定为A，总大小1.16Gb），而仅24％能唯一地映射到亚基因组B（总大小0.63Gb）。
    • 在多倍体和功能冗余的考虑下，玉米应该有很多基因损失。通过与高粱、水稻、二穗短柄草和狗尾草的直系同源基因分析，发现了14%的基因有丢失。
    • 丢失的基因大多与免疫和抗逆有关。

转座元件

  • 以前的版本只有1%的LTR，新版本中有1268M的完整的反转录转座子，还有其他很多的重复。
  • 通过研究重复序列的进化，可以得到物种的进化的相关结果

比较基因组

  • 使用光学图谱，进行比较基因组分析: Ki11 和 W22 ,同B37进行结构比较
    • 由于基因组差异大，Ki11中32%，W22中39% 限制酶切位点，可以比对到B37中。
    • 在着丝粒区域附近，比对较好，表明光学图谱和三代可以很好的解决重复问题。
    • Ki11中32 ，W22中26% 表现出了结构变异，平均大小为20k
    • 说明品系间的差异还是很大的

材料方法：

  • 测三代，使用bionano
  • 组装：
    • PBcR-MHAP和Falcon
    • Quiver修整contig
    • RefAligner合并草图和光学图谱
    • BAC: Mummer
  • 注释：
    • LTRharvest
    • LTRdigest
    • SINE­Finder
    • MGEScan­non­LTR
    • MITE­hunter
    • HelitronScanner
  • 结构分析
    • RefSplit
